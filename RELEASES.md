# Releases

## v0.7.13

- PanSou API URL 按敏感配置处理，WebUI 不再回显真实地址。
- 高级设置移除真实 PanSou 地址占位符，改为已配置状态提示。
- 支持通过 `PANSOU_API_URL` 环境变量或高级设置配置 PanSou API URL。

## v0.7.12

- 订阅卡片简化为海报、标题、TMDB 评分、季号、进度、最后检查时间和保存目录。
- 资源搜索和设置界面聚焦夸克网盘，移除其他云盘类型入口。
- TMDB 元数据刮削会写入剧集总集数，并按当前订阅季号匹配对应季的集数。
- 自动转存目录调整为：电影 `片名（年份）`，连续剧和动画 `剧名（年份）/Season X`。
- 默认订阅季号恢复为第 1 季，电影不使用季号目录。
- 增加订阅新增文件识别、失效状态写入和媒体目录规则测试。

## v0.7.11

- 优化订阅卡片展示，增加 TMDB 评分、季号、进度和最后检查时间。
- 设置页增加订阅检查周期预设和自定义分钟数。
- 重命名模板支持 `{season}` 与 `{season_number}`。
