# v0.5.0 完整更新日志

## 🎉 主要功能

### 1. 手动订阅
- **直接输入链接**：不需要先搜索，直接创建订阅
- **自动嗅探**：检测链接有效性、文件数量、疑似集数
- **密码支持**：支持带密码的分享链接

### 2. 自定义分类管理
- **默认分类**：电影、连续剧、动画（固定）
- **自定义分类**：可添加综艺、纪录片等任意分类
- **独立目录**：每个分类配置独立的保存路径
- **自动转存**：按分类自动转存到对应目录

### 3. 完善的推送系统

#### 7种推送渠道
1. 企业微信机器人
2. WxPusher
3. **Telegram Bot** ⭐ 新增
4. **Bark (iOS)** ⭐ 新增
5. **Gotify** ⭐ 新增
6. **PushPlus** ⭐ 新增
7. **Server酱** ⭐ 新增

#### 4种推送场景
- ✅ **订阅更新** - 发现新资源时推送
- ✅ **订阅失效** - 链接失效时推送
- ✅ **订阅完结** - 标记完结时推送
- ✅ **转存完成** - 夸克转存成功时推送

#### 推送配置
- 场景开关：独立控制每种场景
- 静默模式：支持静默推送（无声音）
- 多渠道：可同时配置多个推送渠道
- 智能消息：自动添加 Emoji，支持富文本

## 📖 配置指南

### Telegram Bot 配置
1. 创建 Bot：@BotFather → /newbot
2. 获取 Token：类似 `123456:ABC-DEF...`
3. 获取 Chat ID：发消息给 @userinfobot

### Bark (iOS) 配置
1. App Store 下载 Bark
2. 复制推送 URL：`https://api.day.app/your_key`

### Gotify 配置
1. 自建服务：https://gotify.net
2. 创建应用获取 Token

### PushPlus / Server酱 配置
- 官网注册获取 Token/Key

## 🚀 使用方式

1. 硬刷新浏览器 (Ctrl+Shift+R)
2. 进入 **⚙️ 系统设置**
3. 配置推送渠道和场景
4. 保存设置
