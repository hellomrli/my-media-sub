my-media-sub 当前架构

当前分支把搜索和挂载同步能力内置到项目中，不再依赖外部 PanSou/OpenList 服务。

  1. 入口：WebUI / 微信文本接口 / HTTP API。
  2. 搜索：search_service 调用内置搜索聚合器，统一结果结构。
  3. 检测：InlineLinkChecker 和 QuarkShareProbe 直接检查夸克分享。
  4. 订阅：subscription_service 规划转存、重命名、去重和通知。
  5. 网盘：quark_save_service 与 routes_quark_drive 直接操作夸克网盘。
  6. 下载：download_service 提交 Aria2 JSON-RPC。
  7. 同步：nas_sync_service 通过本机挂载路径复制到 NAS。
