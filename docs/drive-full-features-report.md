## 📦 技术实现

### 后端 API

**新增接口：**
```
POST /api/quark-drive/move
POST /api/quark-drive/copy
```

**QuarkSaveClient 新方法：**
```python
def move_items(self, fids: list[str], target_fid: str = "0") -> dict
def copy_items(self, fids: list[str], target_fid: str = "0") -> dict
```

**请求模型：**
```python
class QuarkDriveMoveRequest(BaseModel):
    fids: list[str]
    target_fid: str = "0"

class QuarkDriveCopyRequest(BaseModel):
    fids: list[str]
    target_fid: str = "0"
```

### 前端功能

**新增 JS 函数：**
- `updateBatchButtons()` - 更新批量操作按钮状态

- `batchCopyDrive()` - 批量复制

## 🚀 使用指南

### 批量操作
1. 点击右上角 **选择** 按钮
2. 勾选要操作的文件
3. 点击 **批量删除/移动/复制**
4. 对于移动/复制，选择目标文件夹：
   - 输入 `0` - 移动到根目录
   - 输入 `1` - 移动到当前目录
   - 输入 `2` - 移动到上级目录

### 单个文件操作
- **移动** - 点击"移动"按钮，选择目标
- **复制** - 点击"复制"按钮，选择目标
- **下载** - 点击 ⬇️ 按钮，直接下载
- **Aria2** - 点击"Aria2"按钮，发送到 Aria2
- **重命名** - 点击"重命名"按钮，输入新名称
- **删除** - 点击"删除"按钮，移到回收站

### 排序和筛选
- **排序** - 选择排序方式（名称/大小/时间）
- **筛选** - 选择文件类型（全部/文件夹/视频/图片/文档/其他）
